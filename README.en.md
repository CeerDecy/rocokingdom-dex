# Roco Kingdom Encyclopedia

[中文](./README.md) | [English](./README.en.md)

`Roco Kingdom Encyclopedia` is a project focused on organizing and presenting information for *Roco Kingdom: World*, giving players a clearer way to browse creature data and understand attribute matchups.

## Overview

The project currently focuses on two core content areas:

- Creature encyclopedia
- Attribute matchup reference

The site supports both Chinese and English, making it suitable for everyday lookups, attribute research, and long-term community-driven content expansion.

## Main Features

### Creature Encyclopedia

- Browse the list of collected creatures
- Open a dedicated detail page for each creature
- Filter creatures by attribute
- View bilingual names, introductions, habitat information, and evolution paths
- Check each creature's base qualification data

### Attribute Matchups

- View the full attribute matchup system
- Switch between offense and defense perspectives
- Open dedicated pages for individual attributes
- Understand relationships quickly through a visual presentation

### Bilingual Content

- Supports both Chinese and English
- Automatically routes users to the appropriate language version
- Suitable for presenting reference content to users in different languages

### Structured Reference Content

- Organizes creature and attribute data in a unified format
- Provides consistent presentation across list pages, detail pages, and attribute pages
- Makes it easier to expand with more creatures, attribute notes, and related content over time

## Current Coverage

- `42` creature entries
- `18` attribute entries

## Use Cases

- Look up the basic profile of a specific creature
- Quickly understand attribute strengths and weaknesses
- Use it as a reference for team building, battle preparation, and content organization
- Continue growing it as a community-built reference site for *Roco Kingdom: World*
